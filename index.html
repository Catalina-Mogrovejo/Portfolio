BigQuery SQL Project

Conducted a multi-step data exploration and SQL analysis of HUD Point-in-Time Homelessness data using Google BigQuery. Created a custom dataset by extracting and transforming relevant fields, including deriving state codes from CoC numbers using SQL string functions. Analyzed over 2,700 records to identify trends and insights, such as top areas for youth homelessness, changes in unsheltered populations over time, state-level homelessness rankings, and locations excelling in shelter provision. Utilized SQL clauses including WHERE, ORDER BY, GROUP BY, and aggregate functions to answer targeted policy questions, supporting potential resource allocation and program development decisions.
